## 1. 模块概述

### 1.1 模块定位

UI组件模块是编辑器的用户界面层，负责将所有功能以直观、高效的方式呈现给用户。涵盖主编辑器布局、工具栏、资源面板、属性面板、场景视口、状态栏以及各类对话框和辅助组件。

### 1.2 模块职责

- **布局架构**: 三面板加顶部工具栏和底部状态栏的整体布局
- **工具栏**: 提供场景文件操作、编辑命令、对象变换、视图控制和编辑器配置的快捷入口
- **资源面板**: 左侧面板，包含场景层级树、资源浏览器、几何体浏览器和虚拟文件浏览器四个标签页
- **属性面板**: 右侧面板，根据选中对象类型动态展示对应的属性编辑界面
- **多选面板**: 多对象选中时替换属性面板，提供批量操作能力
- **场景视口**: 3D渲染容器，承载三维场景并支持拖拽交互
- **场景覆盖层**: 悬浮于场景之上的辅助控件，包括视角控制、操作提示、性能监控和视图快捷按钮
- **对话框**: 编辑器配置、UserData编辑等弹出式交互界面
- **属性编辑组件**: 49个专用属性面板，覆盖材质、纹理、几何体参数、灯光参数等

### 1.3 核心价值

- 统一的深色主题编辑器界面，降低用户认知负担
- 上下文感知的属性面板，仅展示与当前选择相关的编辑项
- 拖拽驱动的资源加载流程，减少操作步骤
- 可折叠面板布局，灵活适配不同屏幕尺寸
- 实时状态反馈，底部状态栏持续显示关键信息

## 2. 主编辑器布局规格

### 2.1 组件概述

**组件名**: 主编辑器  
**职责**: 整合所有子组件，管理整体布局状态和面板折叠

### 2.2 布局结构

```
+----------------------------------------------------------+
|                    顶部工具栏 (Toolbar)                    |
+------------------+-----------------------+----------------+
|                  |                       |                |
|  左侧资源面板     |     场景视口            |  右侧属性面板    |
|  (ResourcePanel) |   (SceneViewer)       | (PropertyPanel)|
|                  |   + 覆盖层组件          |  或多选面板      |
|                  |                       |                |
+------------------+-----------------------+----------------+
|                    底部状态栏 (EditorFooter)               |
+----------------------------------------------------------+
```

### 2.3 面板折叠机制

**折叠状态存储**:
- 左侧面板折叠状态
- 右侧面板折叠状态
- 底部面板折叠状态

**面板尺寸存储**:
- 左侧面板宽度（像素）
- 右侧面板宽度（像素）
- 底部面板高度（像素）

**切换按钮**:
- 左侧面板切换按钮位于左边缘，垂直居中
- 右侧面板切换按钮位于右边缘，垂直居中
- 按钮样式随面板展开状态变化

### 2.4 应用状态

**响应式状态对象**:
- 加载状态（布尔值）
- 左侧面板折叠状态（布尔值）
- 右侧面板折叠状态（布尔值）
- 底部面板折叠状态（布尔值）
- 当前激活的左侧标签页（字符串）
- 面板尺寸配置（对象）

**左侧标签页选项**:
- 几何体浏览器
- 虚拟文件浏览器
- 资源浏览器
- 场景层级树

### 2.5 状态传递

主编辑器通过依赖注入向子组件提供以下共享状态:
- 对象选择管理器
- 变换操作管理器
- 资源管理器
- 应用状态对象

## 3. 工具栏规格

### 3.1 组件概述

**组件名**: 工具栏  
**职责**: 提供场景文件操作、编辑命令、对象变换、视图控制和编辑器配置的快捷入口

### 3.2 功能分组

#### 3.2.1 文件操作组

| 功能 | 说明 |
|------|------|
| 新建场景 | 清空当前场景所有内容，需二次确认 |
| 暂存到本地 | 将当前场景序列化后保存到本地存储 |
| 保存场景 | 将场景保存到虚拟文件系统 |
| 加载场景 | 从虚拟文件系统加载场景文件 |
| 导出场景 | 将场景导出为JSON文件下载 |
| 导入场景 | 从本地JSON文件导入场景 |

#### 3.2.2 编辑操作组

| 功能 | 说明 |
|------|------|
| 撤销 | 撤销上一步变换操作，无历史记录时禁用 |
| 重做 | 重做已撤销的操作，无重做记录时禁用 |
| 复制 | 复制选中对象，无选中时禁用 |
| 删除 | 删除选中对象，需二次确认，无选中时禁用 |

#### 3.2.3 对象变换组

| 功能 | 说明 |
|------|------|
| 移动工具 | 切换为平移变换模式，无选中时禁用 |
| 旋转工具 | 切换为旋转变换模式，无选中时禁用 |
| 缩放工具 | 切换为缩放变换模式，无选中时禁用 |
| 锁定Y轴 | 锁定变换操作的Y轴，保持水平移动 |

#### 3.2.4 视图控制组

| 功能 | 说明 |
|------|------|
| 聚焦 | 将相机对准选中对象中心，无选中时禁用 |
| 重置相机 | 恢复相机到初始预设位置 |
| 锁定相机 | 禁用相机控制器交互，防止误操作 |

#### 3.2.5 配置组

| 功能 | 说明 |
|------|------|
| 编辑器配置 | 打开编辑器配置对话框 |

### 3.3 快捷键显示

每个工具栏按钮均显示对应的键盘快捷键提示:
- 撤销: Ctrl+Z
- 重做: Ctrl+Y
- 复制: Ctrl+D
- 删除: Delete
- 移动: G
- 旋转: R
- 缩放: S
- 聚焦: F

### 3.4 按钮状态

**禁用条件**:
- 需要选中对象的操作在无选中时禁用
- 撤销/重做在对应栈为空时禁用

**激活状态**:
- 当前变换模式对应的按钮高亮显示
- Y轴锁定按钮在锁定时高亮显示
- 相机锁定按钮在锁定时高亮显示

## 4. 资源面板规格

### 4.1 组件概述

**组件名**: 资源面板  
**职责**: 左侧面板容器，管理四个标签页的切换和内容展示

### 4.2 标签页结构

#### 4.2.1 几何体标签页

展示预定义的基础几何体、扩展几何体、灯光和其他对象。每个项目支持拖拽到场景视口直接创建。

**分类**:
- 基础几何体（立方体、球体、圆柱体、圆锥体、圆环、平面、圆形、圆环等）
- 扩展几何体（十二面体、二十面体、八面体、四面体、圆环结、管状体等）
- 灯光（环境光、平行光、点光源、聚光灯、半球光、矩形面光源）
- 其他对象（相机、组、文本、精灵等）

#### 4.2.2 文件标签页

虚拟文件系统浏览器，展示后端文件库的目录结构和文件。支持拖拽模型文件到场景直接加载。

#### 4.2.3 资源标签页

已加载资源库浏览器，展示已加载的3D模型和纹理。支持文件上传、资源删除、收藏标记。

#### 4.2.4 层级标签页

场景对象层级树，展示场景中所有对象的父子关系。支持搜索、过滤、右键菜单操作。

## 6. 多选面板规格

### 6.1 组件概述

**组件名**: 多选面板  
**职责**: 当选中多个对象时替换属性面板，提供批量操作能力

### 6.2 触发条件

选中对象数量大于1时自动显示，替换右侧属性面板。

### 6.3 批量操作功能

#### 6.3.1 多选变换

创建多选盒（包围盒），通过变换属性面板统一调整多个对象的整体位置、旋转和缩放。

#### 6.3.2 位置对齐

沿指定轴将对象对齐到最小值、中心或最大值位置:
- X轴对齐（最小、居中、最大）
- Y轴对齐（最小、居中、最大）
- Z轴对齐（最小、居中、最大）

#### 6.3.3 平均分布

沿指定轴将对象均匀分布:
- X轴平均分布
- Y轴平均分布
- Z轴平均分布

#### 6.3.4 旋转一致

将多个对象的旋转值统一为第一个对象的旋转:
- X轴旋转一致
- Y轴旋转一致
- Z轴旋转一致

#### 6.3.5 等比例缩放

将多个对象的缩放值统一为最小值或最大值:
- 按最小值等比例
- 按最大值等比例

## 7. 场景层级树规格

**职责**: 以树形结构展示场景中所有对象的层级关系，支持搜索、过滤和右键操作

### 7.2 搜索与过滤

- 按对象名称搜索
- 按对象类型搜索
- 实时过滤，输入即搜索

**过滤选项**:
- 仅显示可见对象
- 仅显示选中对象
- 展开所有节点
- 折叠所有节点
- 刷新层级树

### 7.3 对象操作

**点击选择**:
- 单击选择单个对象
- Ctrl+单击多选
- Shift+单击范围选择

**右键菜单**:
- 复制对象
- 删除对象
- 隔离显示（仅显示该对象）
- 聚焦对象
- 重命名对象

**行内操作**:
- 可见性切换（眼睛图标）
- 锁定状态切换（锁图标）
- 展开/折叠子节点

### 7.4 层级展示

- 缩进表示父子关系
- 选中对象高亮显示
- 锁定对象视觉区分
- 不可见对象视觉区分

## 8. 场景视口规格

### 8.1 组件概述

**组件名**: 场景视口  
**职责**: 3D渲染容器，承载三维场景并处理用户交互

### 8.2 渲染容器

- 自动填充可用空间
- 响应窗口大小变化
- 自动调整相机宽高比和渲染器尺寸

### 8.3 拖拽加载

支持从以下来源拖拽对象到场景:
- 几何体浏览器（创建新几何体）
- 资源浏览器（加载已加载模型）
- 虚拟文件浏览器（加载模型文件）
- 外部文件（通过文件上传）

**放置位置**: 新对象放置到当前相机视点位置

### 8.4 交互处理

- 左键点击选择对象
- 右键点击清空选择
- Canvas自动获取焦点以支持键盘操作

### 8.5 辅助线

- 网格辅助线（可切换显示）
- 坐标轴辅助线（跟随视点目标）
- 线框模式（可切换显示）

## 9. 资源浏览器规格

### 9.1 组件概述

**组件名**: 资源浏览器  
**职责**: 展示和管理已加载的3D模型和纹理资源

### 9.2 标签页

**3D模型标签**:

- 显示缩略图预览

## 12. 场景覆盖层规格

### 12.1 视角控制器

**功能**: 3D方向指示器，点击面切换相机视角

**支持视角**:
- 前视图
- 后视图
- 左视图
- 右视图
- 顶视图
- 底视图
- 等轴测视图

**联动**: 与主相机和控制器实时同步

### 12.2 操作提示

**功能**: 显示当前控制器类型的操作说明

**提示内容随控制器类型变化**:
- 轨道控制器: 左键旋转、右键平移、滚轮缩放
- 地图控制器: 左键平移、右键旋转、滚轮缩放
- 飞行控制器: WASD移动、QE旋转、RF升降、滚轮缩放

**控制器切换**: 内置下拉菜单切换控制器类型

**显示模式**:
- 折叠状态: 仅显示控制器类型名称
- 展开状态: 显示完整操作说明

### 12.3 性能监控

**功能**: 实时显示渲染性能指标

**显示内容**:
- 默认显示FPS数值
- 悬停显示详细性能数据
- 单击切换常驻详细显示

**监控指标**:
- FPS（帧率）
- MS（帧耗时）
- GPU使用情况

### 12.4 视图控制按钮

**功能**: 快捷视图操作按钮

**按钮列表**:
- 重置视图
- 适应屏幕
- 线框模式切换
- 网格显示切换

## 14. 状态管理规格

### 14.1 应用状态

通过响应式对象管理全局UI状态:
- 加载状态
- 面板折叠状态
- 当前激活标签页
- 面板尺寸配置

### 14.2 编辑器配置

通过组合式函数管理持久化配置:
- 控制器类型和参数
- 网格配置
- 坐标轴配置
- 配置持久化到本地存储
- 配置重置功能

### 14.3 状态传递方式

- 主编辑器通过依赖注入向子组件提供共享状态
- 各功能模块通过组合式函数获取独立状态
- 不使用集中式状态管理库

## 16. 使用示例

### 16.1 典型用户工作流

1. 用户打开编辑器，看到三面板布局
2. 从左侧几何体浏览器拖拽立方体到场景视口
3. 立方体出现在场景中心，右侧属性面板自动显示立方体参数
4. 用户在属性面板调整立方体尺寸
5. 用户从资源浏览器拖拽模型到场景
6. 用户Ctrl+单击选择多个对象
7. 右侧面板切换为多选面板，用户执行对齐操作
8. 用户点击底部状态栏相机位置，输入精确坐标
9. 用户通过工具栏保存场景到虚拟文件系统

### 16.2 面板折叠工作流

1. 用户点击左侧面板切换按钮，左侧面板收起
2. 场景视口自动扩展填充可用空间
3. 覆盖层组件自动调整位置以适应新布局
4. 用户再次点击切换按钮，面板恢复展开

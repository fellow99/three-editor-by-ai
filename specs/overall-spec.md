## 1. 产品概述

### 1.1 产品定位

Three Editor by AI 是一个基于 Web 的现代化 3D 场景编辑器，允许用户在浏览器中创建、编辑和管理 3D 场景，无需安装任何桌面软件。

### 1.2 目标用户

- **3D 内容创作者**: 快速原型设计和场景预览
- **Web 开发者**: 在 Web 应用中集成 3D 内容
- **教育工作者**: 3D 可视化和教学演示
- **建筑师/设计师**: 简单的 3D 场景布局和展示

### 1.3 核心价值

- **零安装**: 基于浏览器，无需安装
- **直观交互**: 拖拽式操作，降低 3D 编辑门槛
- **实时预览**: 所见即所得的编辑体验
- **灵活扩展**: 支持多种 3D 格式和后处理效果

## 2. 功能规格

### 2.1 场景管理

#### 2.1.1 场景创建
- 用户可创建新的空场景
- 场景包含默认相机位置和光照设置
- 支持场景背景色/背景纹理设置

#### 2.1.2 场景加载
- 支持从虚拟文件系统加载已保存场景
- 支持从本地文件导入场景（JSON 格式）
- 加载时完整恢复场景状态（对象、材质、动画）

#### 2.1.3 场景保存
- 支持保存到虚拟文件系统
- 支持导出为 JSON 文件下载到本地
- 保存内容包括：所有对象、材质、变换、userData、动画索引

#### 2.1.4 场景导出
- 导出为 glTF/GLB 格式（未来支持）
- 导出为 JSON 格式（当前支持）
- 导出时可选择包含/排除特定对象

---

### 2.2 对象管理

#### 2.2.1 基础几何体创建
用户可创建以下几何体：
- **立方体** (BoxGeometry): 可设置长宽高
- **球体** (SphereGeometry): 可设置半径、分段数
- **圆柱体** (CylinderGeometry): 可设置半径、高度、分段数
- **圆锥体** (ConeGeometry): 可设置半径、高度、分段数
- **平面** (PlaneGeometry): 可设置宽高、分段数
- **圆环** (TorusGeometry): 可设置半径、管径、分段数
- **圆环结** (TorusKnotGeometry): 可设置复杂参数
- **多面体** (Dodecahedron/Icosahedron/Octahedron/Tetrahedron)
- **圆形** (CircleGeometry): 可设置半径、分段数
- **管状体** (TubeGeometry): 可设置路径、半径、分段数

#### 2.2.2 灯光创建
用户可创建以下灯光：
- **环境光** (AmbientLight): 全局照明
- **平行光** (DirectionalLight): 模拟太阳光
- **点光源** (PointLight): 全方向点光源
- **聚光灯** (SpotLight): 锥形聚光灯
- **半球光** (HemisphereLight): 天空/地面颜色渐变
- **矩形光** (RectAreaLight): 矩形区域光源

#### 2.2.3 模型加载
- 支持 GLTF/GLB 格式（含 Draco 压缩）
- 支持 OBJ 格式
- 支持 FBX 格式（未来支持）
- 支持 3D Tiles（基于 3d-tiles-renderer）
- 拖拽文件到场景自动加载到当前视点位置

#### 2.2.4 对象变换
- **移动**: 支持沿 X/Y/Z轴移动
- **旋转**: 支持绕 X/Y/Z轴旋转
- **缩放**: 支持沿 X/Y/Z轴缩放
- **Y 轴锁定**: 可锁定 Y 轴，仅允许 XZ 平面操作
- **撤销/重做**: 完整的历史记录支持

#### 2.2.5 对象选择
- 单击选择单个对象
- 多选支持（Shift+ 点击）
- 框选支持（未来支持）
- 通过场景层级树选择
- 选中对象高亮显示

#### 2.2.6 对象锁定
- 可锁定对象防止误操作
- 锁定对象不可被射线检测选中
- 锁定状态序列化保存

#### 2.2.7 对象删除
- 支持删除选中对象
- 删除前确认（可配置）
- 删除操作可撤销

---

### 2.3 材质系统

#### 2.3.1 材质类型支持
- **MeshBasicMaterial**: 基础材质（不受光照影响）
- **MeshStandardMaterial**: 标准物理材质
- **MeshPhongMaterial**: Phong 高光材质
- **MeshLambertMaterial**: Lambert 漫反射材质
- **MeshPhysicalMaterial**: 物理渲染材质（高级）

#### 2.3.2 材质属性编辑
- **颜色**: 基础色、漫反射色、高光色等
- **纹理**: 漫反射贴图、法线贴图、粗糙度贴图、金属度贴图等
- **参数**: 粗糙度、金属度、透明度、不透明度等
- **高级属性**: 清漆、涂层、透射、厚度等（PhysicalMaterial）

#### 2.3.3 材质管理
- 材质库浏览
- 材质复制
- 材质保存
- 多对象批量应用材质

---

### 2.4 纹理系统

#### 2.4.1 纹理加载
- 支持 JPG、PNG 格式
- 支持 HDR 环境纹理（未来支持）
- 支持 KTX2 压缩纹理
- 拖拽上传纹理文件

#### 2.4.2 纹理编辑
- 纹理偏移 (offset)
- 纹理重复 (repeat)
- 纹理旋转 (rotation)
- 纹理中心 (center)
- 纹理包裹方式 (wrap)

#### 2.4.3 纹理预览
- 纹理缩略图显示
- 纹理 UV 预览（未来支持）

### 2.5 相机控制

#### 2.5.1 控制器类型
- **OrbitControls**: 轨道控制器（环绕观察）
- **MapControls**: 地图控制器（平移 + 缩放）
- **FlyControls**: 飞行控制器（第一人称飞行）

#### 2.5.2 预设视角
- 顶部视图
- 底部视图
- 前视图
- 后视图
- 左视图
- 右视图
- 等轴测视图

#### 2.5.3 相机属性
- 视野 (FOV)
- 近裁剪面
- 远裁剪面
- 相机位置记录与恢复

#### 2.5.4 飞行控制 (FlyControls)
- WASD 键：前后左右移动
- QE 键：上下移动
- 方向键：旋转视角
- 空格键/R 键：向上飞行
- 鼠标：视角旋转
- 速度可调

#### 2.6.1 动画加载

- **SSAO**: 屏幕空间环境光遮蔽（未来支持）
- **DOF**: 景深效果（未来支持）

#### 2.7.3 后处理管理
- 效果启用/禁用
- 效果参数调节
- 效果预设保存（未来支持）

## 3. 非功能规格

### 3.1 性能要求

| 指标 | 目标值 | 测量方式 |
|------|--------|----------|
| 首屏加载时间 | < 3 秒 | 从打开页面到可交互 |
| 帧率 (简单场景) | 60 FPS | 空场景 + 少量对象 |
| 帧率 (复杂场景) | > 30 FPS | 1000+ 对象 |
| 交互响应延迟 | < 100ms | 点击到视觉反馈 |
| 场景加载时间 | < 2 秒 | 10MB 以内场景文件 |

### 3.2 兼容性要求

#### 浏览器支持
- Chrome >= 90
- Firefox >= 90
- Safari >= 15
- Edge >= 90

#### 设备支持
- 桌面电脑（Windows、macOS、Linux）
- 笔记本电脑
- 平板电脑（基础功能）

#### WebGL 支持
- WebGL 2.0（推荐）
- WebGL 1.0（兼容模式）

### 3.3 可用性要求

- 所有功能可通过鼠标和键盘完成
- 快捷键支持常用操作
- 操作有明确的视觉反馈
- 错误提示清晰易懂
- 支持长时间使用（内存稳定）

### 3.4 可维护性要求

- 代码模块化，职责清晰
- 关键逻辑有注释说明
- 公共 API 有 JSDoc 文档
- 遵循项目代码规范
- 符合宪法原则

### 3.5 可扩展性要求

- 支持添加新的几何体类型
- 支持添加新的材质类型
- 支持添加新的后处理效果
- 支持插件化扩展（未来）

## 4. 数据规格

### 4.1 场景文件格式

```json
{
  "version": "1.0",
  "metadata": {
    "createdAt": "2026-04-07T10:00:00.000Z",
    "modifiedAt": "2026-04-07T12:00:00.000Z",
    "author": "User"
  },
  "scene": {
    "background": "#000000",
    "fog": null
  },
  "objects": [
    {
      "uuid": "...",
      "type": "Mesh",
      "geometry": {
        "type": "BoxGeometry",
        "parameters": { "width": 1, "height": 1, "depth": 1 }
      },
      "material": {
        "type": "MeshStandardMaterial",
        "color": "#ff0000",
        "roughness": 0.5,
        "metalness": 0.5
      },
      "transform": {
        "position": { "x": 0, "y": 0, "z": 0 },
        "rotation": { "x": 0, "y": 0, "z": 0 },
        "scale": { "x": 1, "y": 1, "z": 1 }
      },
      "userData": {
        "name": "Box1",
        "animationIndex": 0
      }
    }
  ],
  "lights": [...],
  "cameras": [...]
}
```

### 4.2 userData 规范

userData 字段用于存储自定义属性：

```json
{
  "userData": {
    "name": "自定义名称",
    "description": "对象描述",
    "animationIndex": 0,
    "customProperty1": "值 1",
    "customProperty2": 123
  }
}
```

### 4.3 资源元数据

```json
{
  "name": "Horse.glb",
  "path": "/models/gltf",
  "type": "model",
  "format": "glb",
  "size": 123456,
  "createdAt": "2026-04-07T10:00:00.000Z",
  "thumbnail": "/thumbnails/horse.png"
}
```

## 7. 假设与约束

### 7.1 假设

- 用户具备基本的 3D 概念（坐标、变换、材质）
- 用户使用现代浏览器
- 用户有稳定的网络连接（用于加载资源）
- 用户设备支持 WebGL

### 7.2 约束

- 不支持 VR/AR 设备（当前）
- 不支持协作编辑（当前）
- 不支持实时云同步（当前）
- 移动端功能受限

## 8. 风险与缓解

### 8.1 技术风险

| 风险 | 影响 | 概率 | 缓解措施 |
|------|------|------|----------|
| WebGL 兼容性问题 | 高 | 低 | 降级到 WebGL 1.0，提供错误提示 |
| 大场景性能问题 | 中 | 中 | 实现 LOD、实例化渲染 |
| 内存泄漏 | 高 | 低 | 定期测试，及时清理监听器 |

### 8.2 业务风险

| 风险 | 影响 | 概率 | 缓解措施 |
|------|------|------|----------|
| 用户需求变化 | 高 | 中 | 敏捷开发，快速迭代 |
| 竞品功能超越 | 中 | 中 | 持续创新，聚焦用户体验 |

## 项目介绍

Three Editor by AI 是一个基于 Vue 3 + Three.js 的现代化 3D 场景编辑器，支持三维模型的拖放、调整和属性配置。

### 核心特性

- 🎨 **3D 场景渲染** - 基于 Three.js 的完整 3D 场景管理
- 🧱 **几何体创建** - 15+ 种基础与扩展几何体
- 🔄 **对象变换** - 移动、旋转、缩放，支持撤销/重做
- 📦 **模型加载** - GLTF、OBJ、FBX 等多格式支持
- 🎭 **材质编辑** - 多种材质类型与参数动态配置
- 🎬 **动画系统** - 动画选择、播放、状态导出
- 📁 **虚拟文件系统** - 对接后端文件库
- ✨ **后处理效果** - Bloom、FXAA 等多种后处理

### 技术栈

- **前端框架**: Vue 3.5.17 + Vite 7.0.4
- **3D 引擎**: Three.js 0.182.0
- **UI 组件**: Element Plus 2.10.5
- **样式**: Sass 1.90.0

## 文档维护

### 更新原则

1. **代码变更，文档同步** - 代码 PR 必须包含相关文档更新
2. **准确性优先** - 文档必须与实际代码一致
3. **简洁清晰** - 避免冗余，直击要点
4. **版本管理** - 重大变更标注版本号

### 审查流程

1. **作者自查** - 提交前检查文档完整性
2. **审查人检查** - PR 审查包含文档审查
3. **合并后验证** - 合并后验证文档准确性

### 版本控制

- **MAJOR**: 破坏性变更，文档大版本更新
- **MINOR**: 新功能添加，文档增量更新
- **PATCH**: Bug 修复，文档小修正
